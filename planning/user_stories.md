# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

1. Student Player
A student who creates and manages a football profile, showcases their skills and statistics, and receives invitations to tryouts.
2. Team Captain / Coach
A team leader who searches for players, evaluates player profiles, saves players to a scout list, and invites players to tryouts.
3. Club Organizer
A university football club administrator who manages teams, player registrations, and scouting activities.


## Draft User Stories

Student Player
As a student player, I want to create a football profile so that coaches and team captains can discover me.
As a student player, I want to update my football statistics so that my profile reflects my latest performance.
As a student player, I want to edit my availability so that teams know when I am available for practices and matches.
As a student player, I want to view my scouting status so that I know whether a team is interested in recruiting me.

Team Captain / Coach
As a team captain/coach, I want to browse all registered players so that I can identify potential recruits.
As a team captain/coach, I want to filter players by position, skill level, and availability so that I can quickly find players that fit my team's needs.
As a team captain/coach, I want to save players to my scout list so that I can review them later.
As a team captain/coach, I want to send tryout invitations through a popup modal so that I can invite players without leaving the current page.
As a team captain/coach, I want to add scouting notes to a player so that I can keep track of my evaluations.

Club Organizer
As a club organizer, I want to manage football teams so that player assignments remain organized.
As a club organizer, I want to create, edit, and delete team information so that club records stay up to date.
As a club organizer, I want to view all players and teams in one place so that I can oversee recruitment activities across the club.
